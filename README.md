# CESA-webpage
official webpage for cesa 2022-23

Instruction

step 1:
  open terminal in working directory and run command:
  ```
  git clone <url>
  ```


step 2:
  open directory
  ```
  cd cesa.webpage
  ```
  
step 3:
  install all node modules using
  ```
  npm install
  ```

step 4:
  Run website using
  ```
  npm start
  ```

**ENJOY DEVELOPMENT**
